# **App Name**: KOSH Inventory Dashboard

## Core Features:

- Login Authentication: Secure user login to access the inventory dashboard.
- Inventory Management: Add new products with details (name, description, price, quantity, images) and remove existing ones.
- Real-time Stock Status Updates: Mark products as in or out of stock, instantly reflected on the e-commerce website. The LLM will use the external tool of database lookup to check existing stock data before marking an item out of stock
- Dashboard Overview: Display key metrics: total orders, total inventory items, pending orders, and revenue.
- Order Tracking: View and manage order information, including order status and customer details.
- Customer Management: View customer details such as order history, contact information, and other relevant data.
- Sidebar Navigation: Easy navigation through different sections: Inventory, Orders, Customers, and Home.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and stability.
- Background color: Light Grey (#F5F5F5) for a clean and professional look.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font, providing a modern, machined, neutral feel. It is suitable for both headlines and body text.
- Use consistent, minimalist icons for sidebar navigation and dashboard elements.
- Responsive design with a clean and organized layout to ensure usability on different screen sizes.
- Subtle transitions and animations for a smooth user experience.
